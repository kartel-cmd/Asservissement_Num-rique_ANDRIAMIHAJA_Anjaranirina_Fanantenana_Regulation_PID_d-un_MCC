# Asservissement_Numerique_ANDRIAMIHAJA_Anjaranirina_Fanantenana_Regulation_PID_d-un_MCC
## Description
Ce projet porte sur l'étude et la simulation d'un système d'asservissement numérique, incluant la modélisation, la conception d'un régulateur PID d'un Moteur a courant continu.

## Objectifs
- Modéliser le système
- Concevoir un régulateur PID
- Modeliser le moteur
- Simuler sous MATLAB/Simulink
- Analyser les résultats

## Outils utilisés
- MATLAB
- Simulink

## Structure du projet
- `code/` : scripts et modèles Simulink
- `rapport/` : rapport du projet


## Instructions d'exécution
1. Ouvrir MATLAB
2. Aller dans Simulink
3. Ouvrir le fichier Simulink (`asservissement.slx`) pour visualiser la simulation
